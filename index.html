    Getting Debaly Que Management system License


    1. Open the https://verify.debaly.com in Browser
    2. Sign in with codecanyon.net (envato) Creditial
    3. Press CONFIGURE MAC ADDRESS for Debaly Token - Queue Management System
    4. Add Your MAC Address
      5. Note
      You can add 3 MAC Address
      MAC Address FORMAT - 02:1B:43:13:54:C3
      How to Find MAC Address

    Que Management Client Configuration


    1. Open the que-management-client in IDE
    2. Open the src/environment.js
    3. Replace your debaly license in DEBALY_TOKEN variable
      4.     const API_URL = 'http://localhost:8080';
    const DEBALY_TOKEN = 'REPLACE_YOUR_LICENSE';
                        export {API_URL,DEBALY_TOKEN};

    Que Management Server Configuration


    1. Open the que-management-server in IDE
    2. Open the setting.xml
    3. Replace your license 
          <servers>
        <server>
            <id>github</id>
            <username>github</username>
            <password>REPLACE_YOUR_TOKEN</password>
        </server>
    </servers>

    Database setup


    Connect local database to workbench Link


    Create a database eg: token

    1. Open MySQL connection

    2. Click create schema button

    3. Give a name for database eg: token

    4. Run the query by clicking apply button

    5. Verify the database

    6. Go to the file que-management-server/src/main/resources/application.properties

      spring.datasource.url=jdbc:mysql://localhost:3306/REPLACE_YOUR_DB_NAME?useSSL=false&serverTimezone=UTC&useLegacyDatetimeCode=false&allowPublicKeyRetrieval=true

spring.datasource.username=REPLACE_YOUR_DB_USER
spring.datasource.password=REPLACE_YOUR_DB_PASSWORD

    Run Que Management server in development environment


    1. Go to the que-management-server location in terminal
    2. Type mvn install -s settings.xml in that terminal and press enter
    3. Type mvn spring-boot:run in that terminal and press enter

    Congratulation you successfully setup the Que Management Server

    Run Que Management Client in development environment


    1. Go to the que-management-client location in terminal
    2. Type yarn add or npm install in that terminal and press enter
    3. Type yarn start or npm start in that terminal and press enter

    Congratulation you successfully setup the Que Management Client
